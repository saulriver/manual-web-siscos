1

Para poder ingresar al sistema de información SISCOS se requiere que el usuario se identifique con su nombre de usuario y la clave correspondiente.

Al ingresar el usuario y la contraseña le damos clic en Aceptar para poder ingresar al sistema y si queremos cancelar el proceso se escoge la opción cancelar.

2

Ingresamos las credenciales asignadas,

y damos clic en Aceptar se validará la información. Y permitirá el ingreso al sistema.

Si al ingresar las credenciales no esta asignada ya sea un usuario que no existe o una contraseña nos mostrará un mensaje de información, de Usuario o Clave errada.

3

Posteriormente a la autenticación exitosa,

el sistema automáticamente dirige al usuario a la pantalla inicial del sistema de información SISCOS.

4

La pantalla inicial es el punto más importante para el uso del sistema de información SISCOS,

que le permite al usuario ser orientado de manera personalizada y en tiempo real de las funcionalidades y/o información que se puede llegar a necesitar o reafirmar según sea el caso teniendo control de inventarios y gestión de órdenes de servicios.
